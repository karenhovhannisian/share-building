# share-building

Complexity of algorithm is O(n).

Task:
Дан массив из N долей, представленных в виде N рациональных:
[
'1.5',
  '3',
  '6',
  '1.5'
]
Задача
Написать программу, представляющую эти доли в процентном выражении с точностью до трех знаков после запятой:
[
  '12.500',
  '25.000',
  '50.000',
  '12.500'
]
